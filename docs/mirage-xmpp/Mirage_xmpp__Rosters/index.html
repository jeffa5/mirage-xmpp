Module Mirage_xmpp__Rosters

A roster stores information about contacts for a particular user, it also stores information about subscriptions to the user's presence.

type subscription =
| None

No subscription between the user and the contact

| To

A subscription from the user to the contact

| From

A subscription to the user from the contact

| Both

A mutual subscription between the user and the contact

| Remove

An attribute value in presence stanzas, not for use in the roster
